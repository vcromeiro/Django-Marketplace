# Django Marketplace #

**Django Marketplace** is a simple market place built on Django 2.0.5. Django Marketplace uses Django Admin for administration and Djano Auth for authentication


## Version ##
**0.2.8**


##Installation & Usage##
Deploy Django application

##Supported Browsers##
* Chrome (stable and canary channel)
* Firefox
* IE 9 and 10
* Opera
* Safari
